# Django Retro Market

Проект "Ретро-Маркет" на Django — учебное приложение, имитирующее витрину техники прошлых лет: ретро-смартфоны, игровые консоли, MP3-плееры. Содержит навигацию по категориям, отдельные страницы товаров и блог.

## 🔧 Стек

- Python 3.12
- Django (через Poetry)
- HTML / Bootstrap
- Jinja2-шаблоны
- Class-Based Views (CBV)

## 📦 Установка

1. Клонировать репозиторий:

```bash
git clone https://github.com/Lexish-ru/Module_6.git
cd Module_6
```

2. Установить зависимости через Poetry:

```bash
poetry install
```

3. Применить миграции и запустить сервер:

```bash
poetry run python manage.py migrate
poetry run python manage.py runserver
```

## 🗂 Страницы

- `/` — главная (последние товары)
- `/catalog/` — каталог всех товаров
- `/category/` — список категорий
- `/category/<id>/` — товары в категории
- `/product/<id>/` — карточка товара
- `/contacts/` — форма обратной связи
- `/messages/` — просмотр сообщений
- `/blog/` — список записей блога
- `/blog/post/<id>/` — отдельная запись
- `/blog/post/new/` — создать пост
- `/blog/post/<id>/edit/` — редактировать пост
- `/blog/post/<id>/delete/` — удалить пост

## 🧩 Функции

- Используются `ListView`, `DetailView`, `FormView` и другие CBV.
- Счётчик просмотров записей в блоге.
- Фильтрация только опубликованных постов.
- Общий базовый шаблон (`base.html`) и подшаблон меню (`menu.html`).
- Навигация между разделами.

## 🧪 Заметки

- Контактная форма сохраняет сообщения.
- Bootstrap подключён через CDN.
- Все шаблоны используют `{% extends "base.html" %}`.
